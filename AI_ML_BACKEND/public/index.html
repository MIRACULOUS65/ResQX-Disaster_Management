Disaster Management Backend API

Welcome to the Disaster Management Backend API. This service provides endpoints for disaster prediction and file uploads.

Available Endpoints:

GET
/api/health

Health check endpoint for the Express service

POST
/api/predict

Proxy prediction requests to Flask ML service

Body: JSON with "features" array or "filename" string

POST
/api/upload

File upload endpoint

Body: multipart/form-data with "file" field

Flask ML Service Endpoints:

GET
/health

Health check for Flask ML service

POST
/predict

Direct prediction endpoint on Flask service

POST
/train

Model training endpoint (not implemented)
